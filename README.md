# Prism
Game made for Global Game Jam 2017 at Coleman University.

## Summary
A puzzle game based on additive color theory! Rotate pieces to combine Red, Green, and Blue light into Cyan, Magenta, or Yellow - Combine all three base wavelengths into White light to power the bunny at the end of the stage! Designed with mobile devices as eventual target platform.

## Download
Download this game on the [Global Game Jam website](http://globalgamejam.org/2017/games/prism).
